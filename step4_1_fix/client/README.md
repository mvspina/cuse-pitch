# Syracuse Pitch v7

Adds table visuals and clearer turns.

Includes:
- Deal 7
- Bidding (strictly higher, no equal bids)
- Dealer stuck: if all pass, dealer stuck with 2 and chooses trump
- Discard phase and dealer redeal to 6
- Trick play with follow suit and trump
- Full scoring:
  - High, Low, Jack, Game
  - Set logic for failed bid
  - Shoot The Moon success is instant win
  - Shoot The Moon fail is set for the game target (11 or 21)
- Improved UI:
  - Card table view with seats and center trick
  - Current player highlight

Run:
npm install
npm run dev
